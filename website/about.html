About RunVector

The story behind AI-powered fantasy cricket

What is RunVector?

RunVector is an AI-powered analytics platform designed to help fantasy cricket enthusiasts make data-driven decisions when building their Dream11 teams. Instead of relying on gut feelings or random picks, RunVector uses machine learning to predict player performance based on historical data.

The name "RunVector" comes from the mathematical concept of vectors in machine learning, representing the multidimensional analysis of cricket performance data.

How It Works

RunVector employs advanced machine learning algorithms to analyze ball-by-ball cricket data:

  • Data Collection: Processes Cricsheet format datasets with match-by-match details
  • Feature Engineering: Extracts meaningful patterns like player form, venue performance, opposition matchups, and consistency metrics
  • ML Training: Uses ensemble methods (Random Forest, Gradient Boosting, XGBoost) to learn from historical performances
  • Predictions: Generates fantasy point predictions for selected players based on the context (teams, venue, opposition)

Technology Stack

RunVector is built using modern, production-ready technologies:

  • Python: Core programming language
  • scikit-learn & XGBoost: Machine learning frameworks
  • Pandas & NumPy: Data processing and numerical computation
  • Streamlit: Interactive web application framework
  • Matplotlib: Visualization and data representation

Key Features

  • Dynamic Dataset Upload: Works with any Cricsheet format data
  • On-Demand Training: Train models when you need them, with configurable scope
  • Model Library: Save, manage, and switch between multiple trained models
  • Auto-Save: Never lose your training progress
  • Flexible Workflows: Load pre-trained models or start fresh
  • Smart Analytics: Comprehensive dashboard with player stats and team insights

Open Source

RunVector is completely free and open source. The entire codebase is available on GitHub under an open source license.

Contributions, bug reports, and feature requests are welcome!

About the Creator

RunVector was created by Saurav Kumar, a software developer and cricket enthusiast who got tired of losing money on Dream11 due to poor team selections.

What started as a personal project to combine data science skills with cricket passion has evolved into a full-featured analytics platform.

Disclaimer

RunVector provides data-driven predictions based on historical performance. While the machine learning models are trained on extensive datasets, predictions are not guarantees. Cricket is inherently unpredictable, and player performance can vary due to numerous factors.

Use RunVector as a tool to inform your decisions, not as the sole basis for them. Always apply your own judgment and knowledge of the game.
